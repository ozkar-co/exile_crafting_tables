# exile_crafting_tables

Optional crafting stations for Exile, focused on visuals while keeping full base-game compatibility.

This mod does not replace original stations: it complements them.

## What it includes

Each station uses crafting types that already exist in Exile.

- Restored legacy block:
  - Weaving Frame (`weaving_frame`)
- Adapted legacy block:
  - Hammering Log (`hammering_block`)
- New stations in this package:
  - Crafting Bench (`crafting_spot`)
  - Threshing Sledge (`hammering_block`)
  - Mixing Bucket (`mixing_spot`)
  - Grinding Wheel (`grinding_stone`)

## Compatibility

- Compatible with original Exile.
- Does not add custom recipe types.
- Base stations and base recipes remain available.

## Credits

Thanks to the original creators of Exile and its base mods (`tech`, `nodes_nature`), which this mod builds on for several blocks and behaviors.

In particular:
- `Weaving Frame` keeps behavior inherited from the original system.
- `Hammering Log` is a visual/functional adaptation of a legacy block.
